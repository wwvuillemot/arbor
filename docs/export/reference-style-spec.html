How to build reference.docx in Google Docs
1
Create a new blank Google Doc
Give it a name like reference-template. This file is only for style definitions — it will never contain actual manuscript content.
2
Open paragraph styles
Format › Paragraph styles › Options › Save as my default styles
3
Type a line of sample text for each style
Type one line per style (Normal body text, a Heading 1 title, a Heading 2 chapter name, etc.) — you'll apply the styles to each line individually.
4
Apply and update each style using the settings below
Select the line, set font and size via the toolbar, then in the styles dropdown choose Update "Heading X" to match. Repeat for each style.
5
Download as DOCX
File › Download › Microsoft Word (.docx)
Save as reference.docx in your manuscript directory.
6
Test with Pandoc
Run make docx and upload the output to Google Drive. Verify headings, body text, and blockquotes look right. Adjust reference.docx and rerun if needed.

Makefile target
Makefile make 
# Produces architecting-people.docx for upload to Google Drive
# Run: make docx

PARTS = \
  "PART 0 — INTRODUCTION.md" \
  "PART I — THE SIMULATION MIND.md" \
  "PART II — HUMAN NEEDS.md" \
  "PART III — THE LEADERSHIP MATHS.md" \
  "PART IV — LEADING BEYOND YOURSELF.md" \
  "PART VII — SILICON AND CARBON.md" \
  "PART VIII — REFERENCES.md"

docx:
	pandoc $(PARTS) \
	  --output=architecting-people.docx \
	  --from=markdown \
	  --reference-doc=reference.docx \
	  --toc \
	  --toc-depth=2 \
	  --variable=linestretch:1.25

pdf:
	pandoc $(PARTS) \
	  --output=architecting-people.pdf \
	  --from=markdown \
	  --pdf-engine=xelatex \
	  --template=book.tex \
	  --toc \
	  --toc-depth=2

clean:
	rm -f architecting-people.docx architecting-people.pdf

Before we can talk about leading people, we have to talk about what people are — how the brain constructs reality, why identity is a story we tell ourselves.

EQ > IQ

The expression EQ > IQ does not mean intelligence is irrelevant. It means that above a threshold of competence, emotional intelligence is the stronger predictor of leadership effectiveness.
